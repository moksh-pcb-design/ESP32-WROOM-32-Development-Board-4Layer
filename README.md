# ESP32-WROOM-32-Development-Board-4Layer
4-layer ESP32-WROOM-32 Development Board designed in Altium Designer 25.1 featuring dedicated Ground and Power planes, USB Type-C interface, CH340C USB-UART, AMS1117-3.3V regulator, custom libraries, ERC/DRC verification, BOM, Gerber files, and manufacturing-ready outputs.
